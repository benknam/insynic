# insynic 1.22 版本更新说明

## 🎉 新功能

### 设备级录像设置 Profile
- 每台设备拥有独立的录像配置，可单独保存和加载
- 右键设备列表中的设备项，选择「Record Settings...」打开录像设置对话框
- 包含「连接时启用录像」复选框，勾选后连接设备自动开始录像
- 支持设置输出文件路径、录像格式（MP4/MKV/M4A/MKA/OPUS/AAC/FLAC/WAV）
- 支持选择录制视频流和/或音频流

### 设备级 Streaming Settings Profile
- 每台设备拥有独立的 Streaming Settings 配置，可单独保存和加载
- 右键设备列表中的设备项，选择「Streaming Settings...」即可打开该设备的专属设置
- 配置自动按设备序列号保存，下次连接时自动应用

### 扩展 scrcpy 连接选项
在 Streaming Settings 对话框中新增「scrcpy 选项」分组，包含：
- **镜像时关闭设备屏幕**：开始镜像时自动关闭设备屏幕，节省电量
- **保持设备唤醒**：连接时防止设备进入休眠
- **连接时点亮屏幕**：建立连接时自动点亮设备屏幕
- **禁用屏幕保护**：设备连接时禁用电脑屏幕保护
- **启用控制（鼠标/键盘）**：允许使用鼠标和键盘控制设备

### 屏幕传输音频功能
- 在 Streaming Settings 中新增「音频设置」分组
- 支持启用/禁用音频传输
- 支持设置音频比特率（64/128/192/256 kbps）
- 支持选择音频编码器（OPUS/AAC）
- 支持选择音频源（输出/麦克风/播放）

### 设备列表右键菜单增强
- 右键设备项：显示「Streaming Settings...」 + 「Record Settings...」 + 「Connect All Devices」
- 右键空白处：显示「Connect All Devices」
- 右键网络设备：额外显示「Disconnect Network Device」选项

## 🔧 优化改进

- 移除系统顶部菜单栏的「Streaming Settings」菜单项，统一由设备右键菜单入口
- 修复中文语言下，已连接设备列表中设备状态颜色显示错误（橙色→绿色）
- 优化窗口层级管理：每个设备窗口组（屏幕+侧栏+控制栏）拥有独立层级，仅在激活时置顶
- 切换到其他应用时，所有窗口正常被遮挡，不再遮挡其他应用窗口

## 🐛 问题修复

- 修复断开连接时可能导致的程序崩溃
- 修复长时间运行后屏幕窗口卡死问题
- 修复语言切换后界面不立即更新的问题
- 修复文件管理器按钮文字不随语言切换的问题
- 修复网络连接对话框取消后设备选择异常的问题
- 修复录像功能初始化时的空指针回调导致崩溃问题
- 修复录像功能清理顺序错误导致崩溃问题
